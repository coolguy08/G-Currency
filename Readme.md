# G-Currency
## _A Free API for Getting Currency Rates_

An API that uses Web Scraping to Get Live Currency Rates



## `Features`
- NO API Key Required
- Live Currency Rates

## `Tech`
- Web Scraping 
- Nodejs
- Cheerio
- Express

## `Usage`
### Endpoints
* /usage to get Usage details
* /getrates to get currency rates
